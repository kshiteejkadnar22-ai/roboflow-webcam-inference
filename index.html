Roboflow Webcam Inference

Real-time object detection using Roboflow Inference SDK

Project Overview

This project demonstrates a real-time object detection system using Roboflow's WebRTC-based inference pipeline.

System Architecture

  • Frontend: GitHub Pages (HTML, CSS)
  • Backend: Python + Roboflow Inference SDK
  • Model: Custom Roboflow-trained model

How It Works

  1. Webcam captures video frames
  2. Frames are streamed via WebRTC
  3. Roboflow performs inference
  4. Predictions are returned with annotations
